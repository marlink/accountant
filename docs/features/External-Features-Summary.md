# External Features Summary

## Akaunting
- Bank accounts, deposits/transfers
- Multi‑company
- Reporting
- Client portal
- Recurring invoices/bills/payments
- Customer/vendor management, summaries
- Tax rates, multi‑currency, discounts
- Attachments (receipts, evidence)
- Transaction categories
- Bulk actions
- Dashboards/widgets

## ERPNext (Accounting)
- Flexible Chart of Accounts (tree), multi‑company, multi‑currency
- Double‑entry general ledger; drill‑down
- AP/AR: invoicing, payments, aging, follow‑ups
- Taxes & compliance via templates (GST/VAT, etc.)
- Cost centers, dimensions, budgeting
- Financial statements: GL, Trial Balance, Balance Sheet, P&L, Cash Flow
- Assets with depreciation schedules (SL/WDV/DDB)
- Account & bank reconciliation tools

## Open Accounting
- Double‑entry system with customizable accounts
- Dashboard, accounts, transactions editor
- Basic reports
- REST API for users/orgs/transactions/accounts/prices

## Overlap & Signals
- Core: double‑entry + Chart of Accounts + GL/TB (all three)
- Operational: invoicing/AP/AR + taxes + multi‑currency (Akaunting, ERPNext)
- Bank feeds & reconciliation (ERPNext, Akaunting)
- Usability: attachments/OCR, dashboards, client portal, bulk actions (Akaunting)
- Planning/analysis: cost centers/dimensions, budgeting (ERPNext)
- Advanced finance: assets & depreciation (ERPNext)
