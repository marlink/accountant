# Phase 2 Features Specification

## Scope
Usability, automation, and planning features to improve productivity.

## Features
- Client Portal
  - Customer login to view invoices/statements; download PDFs
  - Online payments (bulk, single); payment confirmations
- Recurring Items
  - Schedules for invoices/bills/revenues/payments; calendar view
  - Auto‑generation and notification; skip/override rules
- Cost Centers & Dimensions
  - Tag transactions with dimension values (e.g., channel, region)
  - Profitability reports per dimension; filters across reports
- Budgeting
  - Define budgets per cost center/dimension; monthly splits
  - Alerts on overspend; simple variance reports
- Dashboards & Widgets
  - Configurable widgets (cash, AR/AP, revenue, expenses, FX)
  - Shareable dashboards; role‑based visibility
- Bulk Actions & Saved Views
  - Enable/disable/export/delete in batch; saved filters
- Discounts
  - Line‑level or invoice‑level discounts; audit

## Non‑Functional
- Email/webhooks for recurring generation and portal actions
- Role‑based access for dashboards and portal

## Deliverables
- UI wireframes: Client Portal, Recurring, Dashboards
- Report definitions: Dimension profitability, Budget vs Actual
