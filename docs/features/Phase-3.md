# Phase 3 Features Specification

## Scope
Advanced finance and enterprise‑grade controls.

## Features
- Assets & Depreciation
  - Asset register; methods: Straight Line, WDV, Double Declining Balance
  - Auto journal for acquisition/depreciation/disposal
- Consolidation & Inter‑company
  - Cross‑company eliminations; consolidated BS/P&L/Cash Flow
  - Inter‑company sales/purchase with mirrored postings
- Approval Workflows
  - Multi‑level approvals for bills, payments, journals
  - Audit trail of approvals; role constraints
- Advanced Permissions
  - Fine‑grained roles, data scopes (company, dimension), print formats

## Non‑Functional
- Compliance exports; localization hooks
- Performance targets maintained under consolidation loads

## Deliverables
- Workflow diagrams; consolidation rules spec
- Report definitions: consolidated statements; eliminations
